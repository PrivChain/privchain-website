# PrivChain Overview

PrivChain is privacy infrastructure for machine-native payments.

## Public mission

- Integrate with x402-compatible payment semantics.
- Provide private settlement and verification infrastructure.
- Offer developer-ready surfaces across API, SDK, wallet signing, and MCP tools.

## Public stage signal

- Private testnet hardening is active.
- Interface names and integration boundaries are public.
- Sensitive internals remain intentionally private until release gates are complete.
- Optional analytics is consent-gated; default website mode is essential-only.
