Glossary of Core Terms

Definitions used across protocol specs, API docs, and operational runbooks.

Commitment

Cryptographic value that binds hidden transaction state.

Facilitator

Service that verifies payloads and drives settlement lifecycle.

Health Check

Service endpoint used for availability and readiness probes.

Idempotency

Property ensuring repeated settlement requests resolve consistently.

MCP Tooling

Agent-facing tool interfaces for pay, lookup, and reconciliation actions.

Nullifier

Replay-protection value preventing reuse of proof payloads.

Payment Scheme

Identifier that defines payload format and verification semantics.

Proof Payload

Serialized proof and metadata bundle supplied for verification.

Reconciliation

Process that updates long-pending settlement records.

Remote Signer

External signing service used instead of local private key handling.

Settlement Reference

Identifier returned from settle used for lifecycle lookup.

x402 Compatibility

Support for HTTP payment semantics while extending with privacy controls.
